# Tutor+

**Приложение для частных репетиторов, преподавателей, коучей, психологов и тренеров** — единый инструмент для расписания, учёта учеников/клиентов, финансов, аналитики и напоминаний. Интерфейс на русском, светлая и тёмная темы.

> 🚧 **В активной разработке.** **Фаза 2 (финансы + аналитика) реализована** — поверх ядра домена (Ученики/Клиенты, Расписание, «Сегодня») добавлены **Финансы** (операции tri-state, погашение долга, периоды, CSV-экспорт) и **Аналитика** (графики дохода/занятий, доли направлений, задолженности, сравнение периодов) — всё на офлайн-хранилище WatermelonDB с dual-mode. **[Живое веб-превью →](https://kengston.github.io/tutor-plus-app/)**

## Идея

Частные практики теряют время на организацию расписания, учёт оплат и напоминания, а единого простого инструмента нет. Tutor+ закрывает весь операционный контур в одном мобильном приложении.

Ключевая механика — **dual-mode**: тип практики выбирается в профиле и меняет терминологию во всём приложении:
- режим **«Ученик»** — урок, предмет, ставка за урок (репетиторы, подготовка к ЕГЭ/IELTS/TOEFL);
- режим **«Клиент»** — встреча, направление, ставка за встречу (коучи, психологи, тренеры).

## Возможности

- **Сегодня** — сводка дня, ближайшее занятие, быстрые действия, свайп-жесты.
- **Расписание** — календарь и список, фильтры, свободные окна.
- **Ученики/Клиенты** — карточки, статусы, долги, быстрая связь.
- **Финансы** — операции tri-state (оплачено/долг/ожидается), погашение долга, периоды (неделя/месяц/год/произвольный), сводки и группировка по датам, способы оплаты, CSV-экспорт.
- **Аналитика** — доход по месяцам, KPI (занятия/отмены/средний чек), доли и топ направлений, динамика по неделям, задолженности с переходом к ученику, сравнение периодов.
- **Уведомления** — напоминания о занятиях и оплатах, настраиваемое время.

## Технологии

- **Клиент:** [Expo](https://expo.dev/) (React Native + TypeScript) — единый код на **web · Android · iOS**.
- **Бэкенд:** [Supabase](https://supabase.com/) (PostgreSQL + Auth + Storage).
- **Офлайн-first:** [WatermelonDB](https://watermelondb.dev/) — локальное хранилище и синхронизация.
